6. How the Map Was Built (GroundingDINO Pipeline)

Step 1 — Goal Decomposition: Input "find refrigerator" → detection classes: refrigerator, door, cabinet, chair, table, desk, sofa, sign, fire extinguisher, plant, printer
Step 2 — GroundingDINO Detection: Each photo sent to GroundingDINO SwinT with text prompt "refrigerator . door . cabinet . ..." → returns bounding boxes + confidence scores (threshold: box=0.30, text=0.25, top 5 kept)
Step 3 — Object Indexing: Build reverse index: for each object type, list all photos where it appears with scores
Step 4 — Photo Connections: Consecutive photos compared by Jaccard similarity of detected object sets. High similarity = same area.
Step 5 — Zone Assignment: Photos grouped into zones by sequence and shared objects. Each zone becomes a node in the topological map.
Step 6 — Navigation Path: Shortest path from current zone (Main Lobby, node 1) to target zones (Coffee Station node 2, Fridge Area node 8) using NetworkX.
Step 7 — Confidence Tracking: As user walks toward the fridge, bounding box grows larger and score increases → UniGoal interprets growing box as "getting closer" and triggers ARRIVED when score > threshold and box fills >40% of frame.
